2  Operadores Lógicos e Operadores de Comparação

Em resumo, operadores lógicos são utilizados em Python e R para realizar operações de lógica booleana em expressões condicionais. Eles ajudam a determinar a verdade ou falsidade de uma condição. Operadores de comparação são usados em Python e R para comparar valores e expressões, retornando valores booleanos (True ou False) que indicam se a comparação é verdadeira ou falsa.

2.1 Tabela Verdade

Antes de mais nada, vamos precisar remotar conceitos presentes na Tabela Verdade.

2.1.1 Conectivo “E”

P Q RESULTADO P E Q
VERDADEIRO VERDADEIRO VERDADEIRO
VERDADEIRO FALSO FALSO
FALSO VERDADEIRO FALSO
FALSO FALSO FALSO
Fonte: DevSci

Condição para conectivo “E” ser verdade deve “VERDADEIRO VERDADEIRO” e vice-versa.

2.2 Operadores lógicos

  1. &: Retorna verdadeiro se ambas as expressões forem verdadeiras;

  2. |: Retorna verdadeiro se pelo menos uma das expressões for verdadeira;

  3. !: Inverte o valor de verdade ou falsidade de uma expressão.

Esses operadores lógicos são amplamente utilizados em estruturas condicionais (como if, elif/else em Python e if, else em R) para controlar o fluxo do programa com base nas condições especificadas.

2.3 Operadores de comparação

  1. ==: Igual a (verifica se dois valores são iguais);

  2. != ou <>: Diferente de (verifica se dois valores são diferentes);

  3. <: Menor que (verifica se o valor à esquerda é menor que o valor à direita);

  4. >: Maior que (verifica se o valor à esquerda é maior que o valor à direita);

  5. <=: Menor ou igual a (verifica se o valor à esquerda é menor ou igual ao valor à direita);

  6. >=: Maior ou igual a (verifica se o valor à esquerda é maior ou igual ao valor à direita).
